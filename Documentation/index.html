  • How do i upload the template?

    You can simply drop all the folder "COMMITY1.1" into your ftp from your website.

    If you don't have a program for uploading it, i would suggest you to download FileZilla ( https://filezilla-project.org/ ) It's an open source software free of charge under the terms of the GNU General Public License.


    Here is a tutorial for FileZilla : http://www.siteground.com/tutorials/ftp/filezilla.htm

  • How do i edit the content?

    All the content are on the HTML File ( index.html ).


    This theme is a one page portfolio. All of the information within the main content area is nested within a div with an id of "primaryContent". The general template structure is the same throughout the template. Here is the general structure looks like.





    As you can see, it's pretty well commented, you'll found easy what you are looking for with a simple CTRL+F ( or CMD+F on Apple Keyboard ) on your edit software.


    if you are comfortable with HTML&CSS, you can have take a look on the Basic Gumby Grid framework ( http://www.gumbyframework.com ), you'll found how to modify correctly the width of the different element into your future website.

  • I want a full height header and a background

    Search for the #header from the main stylesheet ( style.css ) line 331



    For a full 100% Height header, just put a "height:100%;".



    For adding a background image add this line : "background-image:url('../img/yourimage.jpg')"

  • How to recieve the e-mail from the visitor?

    Just replace the e-mail on line 11 from the from_sender.php file by your e-mail.

  • How work the scroll appear effect?

    I'm using the scrollMe plugin by Nick Pearson


    The usage of the plugin is pretty well detailed on the page of the plugin : http://scrollme.nckprsn.com/.

  • How do a add a scrollTo on a link?

    Simply add the "goto" class on your link.



  • Sources

  • Javascripts Files
    • - jQuery is a Javascript library that greatly reduces the amount of code that you must write


    • - Modernizr make an altenative for others navigators if they don't accept some code


    • - main.js is the main javascript file that call the differents plugins


    • - contact_form.js is used for the newsletter from the footer section


    • - classie.js is used for reduced height of the navigation when you scroll down


    • - jquery.flexslider.js is used for slider of clients section and agency section.


    • - jquery.parallax-1.1.3.js is used for the header background, which you can set up into the main.js file.


    • - mediaCheck.js allows you to disable or enable javascript code into the main.js


    • - waypoints.min.js is used in this case for the scrolling effect.


    • - jquery.scrollto.js is used for the smooth scrolling from the navigation.


    • - jquery.scrollme.js is used for the animations for each elements when they entering into the viewport. You can found how to set up a transition here : http://scrollme.nckprsn.com/


    • - yui.js is used for the header text which is scrolling with the viewport.
  • CSS Files
    • - style.css is the main stylesheet that contains all of the specific stylings for the page.


    • - gumby.css is the framework that i'm using. you have nothing to touch there. It's for the row and column.


    • - font-awesome.min.css is the stylesheet used for the social icon.


    • - flexslider.css is the stylesheet used for slider of clients section and agency section.
